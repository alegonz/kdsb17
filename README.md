##### Requirements
* Python 3
* Keras 1.2.2
* Theano 0.8.2
* Numpy 1.12.1
* Scipy 0.19
* pydicom 0.9.9
* CUDA Version 7.5.18

Data pre-processing performed using the Kaggle docker image.
Deep learning models were trained in a Amazon Web Services p2.xlarge instance running the Deep Learning AMI Amazon Linux (version 2.0).
